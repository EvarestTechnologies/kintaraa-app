# Kintaraa — GBV Support Platform

Kintaraa is a mobile-first platform that connects Gender-Based Violence (GBV) survivors with the service providers who can help them. It replaces fragmented, manual referral processes with a coordinated, digital system built around survivor safety and provider effectiveness.

## Current Status

The platform is in active development. As of March 2026:

- **Frontend:** 98% complete (React Native, iOS + Android)
- **Backend:** Django REST API operational, connected to mobile app for authentication and incident reporting
- **Integration:** Provider dashboards in active development; core case assignment system functional

## Quick Facts

- **Platform type:** Mobile application (iOS + Android) + Web dashboard (planned)
- **Primary users:** GBV survivors and 7 categories of service providers
- **Technology stack:** React Native, Expo, Django, PostgreSQL
- **Deployment region:** Designed for Kenya; adaptable for any region
- **Data location:** Provider-configurable; supports local and cloud deployment
